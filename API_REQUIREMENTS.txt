# API Dashboard Dependencies

Add these packages to your requirements.txt or install them manually:

## Installation

```bash
pip install aiohttp>=3.8.0 aiohttp-cors>=0.7.0
```

## Details

### aiohttp (v3.8.0+)
- Async HTTP client/server library
- Used for REST API server
- Provides web framework for dashboard backend

### aiohttp-cors (v0.7.0+)
- CORS (Cross-Origin Resource Sharing) support
- Allows dashboard to make requests across origins
- Needed for mobile/remote access
